# Research: Security Policy Check

## 목적

사내에서 자기진화형 AI 에이전트를 호스팅할 수 있는지, 특히 bash 실행 권한과 외부 LLM API 호출이 허용 가능한지 사전 확인한다.

## 핵심 질문

- [ ] 자율 bash 실행 권한을 가진 AI 에이전트 호스팅이 가능한가
- [ ] 외부 LLM API 호출에 대한 사내 정책은 무엇인가
- [ ] 사용자별 격리 수준에 대한 사내 기준은 무엇인가
- [ ] 데이터 잔존/삭제 정책은 무엇인가
- [ ] Phase 1 PoC에서 허용 가능한 최소 범위는 어디까지인가

## 인터뷰/확인 대상

- [ ] 보안팀
- [ ] IT/인프라 담당자
- [ ] 개인정보 또는 컴플라이언스 담당자

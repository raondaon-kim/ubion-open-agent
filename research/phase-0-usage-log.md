# Phase 0 Usage Log

## 목적

1주간 Hermes를 실제 업무에 사용하면서 세션별 효과, 실패 패턴, skill/memory 변화 조짐, 비용 신호를 기록한다.

## 주간 목표

- [ ] 최소 5일 사용
- [ ] 총 10세션 이상 수행
- [ ] 작업 축 3개 모두 사용
- [ ] 각 작업 축 최소 2회 이상 사용

## 작업 축 정의

| 작업 축 | 설명 | 성공 기준 |
| --- | --- | --- |
| 반복성 높은 실무 작업 | PDF 파이프라인 관련 반복 작업 | 사람이 직접 후처리하는 시간을 줄이고 재사용 가능한 출력이 나와야 함 |
| 창의/탐색 작업 | 시 에이전트 관련 창작/탐색 작업 | 피드백이 다음 결과에 반영되고 취향 적합도가 개선되어야 함 |
| 장기 컨텍스트 작업 | Genlearn 관련 누적 맥락 작업 | 며칠에 걸친 결정과 문맥을 기억하고 이어서 작업해야 함 |
